## Part 5: Tracking Dashboard Setup

### Option A: Google Analytics 4 (Free)

**Track affiliate click-throughs:**

Add GA4 event tracking to all outbound affiliate links:
```javascript
// Add to your layout or tracking component
document.addEventListener('click', (e) => {
  const link = e.target.closest('a[href*="clio.com"], a[href*="mycase.com"], a[href*="practicepanther"]');
  if (link) {
    gtag('event', 'affiliate_click', {
      'software_name': link.dataset.software || 'unknown',
      'page_path': window.location.pathname,
      'link_url': link.href
    });
  }
});
```

**Or using Next.js data attributes:**
```tsx
<a 
  href={affiliateUrl} 
  data-software={software.name}
  onClick={() => trackAffiliateClick(software.name)}
  target="_blank"
  rel="noopener noreferrer"
>
  Try {software.name} Free
</a>
```

Create a GA4 custom report:
- Go to **Reports â†’ Custom â†’ Create Report**
- Dimension: Page path
- Metric: Event count (affiliate_click)
- Filter: Event name = affiliate_click
